经典的快速排序

在lambda函数里进行递归，根本不是人，基本结构是将一个列表拆成比第一个元素小的和比第一个元素大的两个列表，两个列表分别排序之后，拼在一起

  1. 思路是拆成两个部分，但其实是三个列表相加，这样可以保证每一次都有变化
  2. 这个终止条件在列表循环里也经常用
[1, 1, 2, 2, 3, 4]
